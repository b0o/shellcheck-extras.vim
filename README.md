# shellcheck.vim

## Install

```vim
NeoBundle 'kawaz/batscheck.vim', {'depends': ['scrooloose/syntastic']}
```

## Commands

- `:ShellcheckSuppressWarnings` - Suppress warnings on current line. It support range.
